# kickskater

## Usage

[install choreonoid (version 2.2)](https://choreonoid.org/ja/documents/2.2/index.html#)

```
git clone git@github.com:Actat/kickskater.git
cd kickskater
make
choreonoid KickSkater.cnoid
```
